rbc.errors¶

Description

rbc-specific errors and warnings.

exception rbc.errors.ForbiddenIntrinsicError¶

Raised when the user uses an intrinsic with name in a blacklist. For more info, see: https://github.com/xnd-project/rbc/issues/207

with_traceback()¶

Exception.with_traceback(tb) – set self.__traceback__ to tb and return self.

exception rbc.errors.ForbiddenNameError¶

Launch when the user defines a function with name in a blacklist. For more info, see: https://github.com/xnd-project/rbc/issues/32

with_traceback()¶

Exception.with_traceback(tb) – set self.__traceback__ to tb and return self.

exception rbc.errors.OmnisciServerError¶

Launch when OmnisciDB server raises a runtime error that RBC knows how to interpret.

with_traceback()¶

Exception.with_traceback(tb) – set self.__traceback__ to tb and return self.

exception rbc.errors.UnsupportedError(msg, loc=None, highlighting=True)¶

Launch when an attempt is to use a feature that is not supported for a given target.

add_context(msg)¶

Add contextual info. The exception message is expanded with the new contextual information.

patch_message(new_message)¶

Change the error message to the given new message.
